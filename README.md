# Earendel Portfolio Dashboard

Private investment dashboard. Password protected.

## Deploy to GitHub Pages

1. Push this folder to a GitHub repo
2. Go to Settings → Pages → Source: Deploy from branch → `main` → `/` (root) or `/dashboard`
3. Your dashboard will be live at `https://yourusername.github.io/repo-name`

## Local preview

Open `index.html` directly in any browser — no server needed.
